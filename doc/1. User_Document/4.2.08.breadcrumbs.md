# 브레드크럼(Breadcrumbs)

네비게이션 계층 내에서의 현재페이지 위치를 표시하는 컴포넌트.

> 구분자(/)는 자동적으로 CSS 내에서 `:before`와 `content`를 통해 추가 된다.

```
<ol class="breadcrumb">
  <li><a href="#">Home</a></li>
  <li><a href="#">Library</a></li>
  <li class="active">Data</li>
</ol>
```

## Snippet
```
<ol class="breadcrumb">
  <li><a href="#">Home</a></li>
  <li><a href="#">Library</a></li>
  <li class="active">Data</li>
</ol>
```
